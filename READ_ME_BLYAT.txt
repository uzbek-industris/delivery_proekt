Проект самодостаточен, по идее должен заработать из коробки.

Сделайте только "/delivery-venv/Scripts/activate" , чтобы работать с виртуальным окружением проекта (имеет на борту питон 3.10; если есть VS Code, то должен это отобразить)

Запускать его с cmd командой "pythom manage.py runserver" через cmd

в html файлах делаете "{% static "images/картинка" %}" или "{/static/images/картинка" %}", как вам удобно;
для css файлов прописывать полный путь "/static/images/картинка".

Пока всё. Базы подтяну попозжа
